Ultra-Fast Port Scanner

  • Tech Stack: C, DNS, Networking
  • Github URL: Project Link

A traditional port scanner has a particular task: to scan ports. Most commonly these are TCP ports on a device or a website with an IP address. Port scanners are particularly useful for network engineers, security analysts and basically anyone who wants to play around with websitetes and computers. They work the following drawbacks that are overcome by this scanner:

  • They are incredibly slow! (2 minutes / IP).
  • They can only scan only one IP/URL at a time.
  • The syntax is not very straight forwars.
  • Non-availability of source code.

    • Presented program scans URL open ports upto 10X faster than traditional scanners by using upto 100 of child scanners concurrently. The scanner also lists all IPv4 and IPv6 addresses allotted to each URL.
